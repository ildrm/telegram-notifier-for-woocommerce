Send WooCommerce order notifications to messaging channels and groups.

== Description ==
WooCommerce Message Notifier allows you to receive instant notifications about your WooCommerce orders through messaging channels. Stay updated about new orders and status changes in real-time.

== Installation ==
1. Upload the plugin files to wp-content/plugins/wc-message-notifier
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings under Settings > Message Notification

== Frequently Asked Questions ==
= Does this plugin require WooCommerce? =
Yes, WooCommerce must be installed and activated for this plugin to work.
